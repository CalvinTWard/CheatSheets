Display and Positioning

Display
display: inline; Inline elements take up only as much space as needed. Allows multiple elements on the same line.
display: block; Displays content on its own line in its own block.
display: inline-block; A block that that can display next to other blocks.
Position
position: static; The element is positioned according to the normal flow of the document. This is the default value.
position: relative; Positions an element relative to its static position, and then offset using the top, bottom, right, and left properties.
position: fixed; Removes the element from the flow. The element stays on screen at all times.
position: sticky; Makes an element stay in place until it reached a certain position on screen, and would then stay on screen while scrolling.
position: absolute; All other elements will act like it is not present.
Float
float: right; Places the element to the far right of its container, allowing text and inline elements to wrap around it.
float: left; Places the element to the left side of the container. Float removes the element from the normal flow of the page, though still remaining a part of the flow.
